**Informações do bot**

PREFIX: c!

- EVENTOS:

[x] welcome - mensagem de boas vindas;

[x] reaction role - atribuir cargo de membro assim que o usuario novo concordar com as regras;

[x] member count - contagem de membros do servidor.

*mais funções em breve.*

- COMANDOS:

[x] ping - função de latencia entre servidor do bot e api discord;

[x] kick - Esse comando é para chutar um usuário do servidor. Você pode colocar uma razão para está chutando um usuário do servidor;

[x] ban - Esse comando é para banir um usuário do servidor. Você pode colocar uma razão para está banindo um usuário do servidor;

[x] clear - Limpa as mensagens de acordo com a quantidade estipulada;

[x] mute/unmute - Esse comando é para mutar um usuário do servidor;

[x] music bot - em processo de implementação do comando de musica do bot;

[] roles - determina cargos para cada usuário abaixo do cargo do bot;

[] chat private - gerenciamento de chat privado.

*mais comandos em breve*
